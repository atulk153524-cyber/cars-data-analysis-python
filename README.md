🚗 Cars Data Analysis (Python)
📌 Overview

This project focuses on analyzing a cars dataset using Python and Pandas to uncover meaningful insights about car manufacturers, pricing, fuel efficiency, and performance.

The dataset includes detailed information about various car models such as engine size, horsepower, mileage, weight, and pricing. The analysis is performed in a Jupyter Notebook with visualizations to better understand trends and relationships.

🛠️ Tools & Technologies

Python

Pandas

Matplotlib

Jupyter Notebook / Google Colab

📂 Dataset

📌 Key Insights

Sedan is the most common car type in the dataset.

Higher horsepower generally leads to lower fuel efficiency.

Cars from the USA tend to have larger engines.

Strong positive correlation between MPG_City and MPG_Highway.
