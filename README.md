# cars-data-analysis-python

## Overview
This project analyzes a cars dataset using Python and Pandas to extract insights about car manufacturers, pricing, fuel efficiency, and performance.
The dataset contains information about different car models including their manufacturer, origin, engine size, horsepower, fuel efficiency, weight, and price. The analysis was performed using Python, Pandas, and Matplotlib.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

## Dataset
The dataset contains information about different car models including:
- Manufacturer
- Model
- Origin
- Engine Size
- Horsepower
- Fuel Efficiency (MPG)
- Weight
- MSRP and Invoice prices

## Analysis Performed
The project includes the following analysis:

- Data exploration
- Checking missing values
- Price analysis
- Fuel efficiency analysis
- Car type distribution
- Filtering cars based on cylinders
- Correlation analysis
- Data visualization (bar chart, scatter plot, histogram)

## Key Insights
- Sedan is the most common car type in the dataset.
- Higher horsepower generally leads to lower fuel efficiency.
- Cars from the USA tend to have larger engines.
- There is a strong positive correlation between MPG_City and MPG_Highway.
